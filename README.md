## 📋 Kanban Board

We use GitHub Issues with labels for kanban-style task management.

### Board Columns (Labels)

| Column | Label | Color |
|--------|-------|-------|
| Backlog | `kanban::backlog` | Gray |
| Ready | `kanban::ready` | Green |
| In Progress | `kanban::in-progress` | Yellow |
| Review | `kanban::review` | Purple |
| Verification Required | `kanban::verification` | Orange |
| Done | `kanban::done` | Green |
| Blocked | `kanban::blocked` | Red |

### Priority Labels

| Priority | Label |
|----------|-------|
| Critical | `priority::critical` |
| High | `priority::high` |

### Verification Label

| Type | Label |
|------|-------|
| Two-person verification required | `verification::required` |
